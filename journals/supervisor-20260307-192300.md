## Suggested Improvement

1. **Sequential merge with conflict resolution**: When agents touch overlapping files, merge them one at a time and attempt basic conflict resolution (e.g., taking both sides for additive changes, deduplicating identical additions).
2. **Dependency-aware scheduling**: The planner should declare file-level dependencies between subtasks. Subtasks touching the same files should be serialized — later subtasks branch from the integration branch AFTER earlier ones are merged.
3. **Merge order optimization**: When all agents complete, analyze git diff overlap and merge the most comprehensive agent first, then skip agents whose changes are pure subsets.
4. **Supervisor auto-escalation**: Instead of marking all subtasks as failed immediately, spawn a supervisor agent to attempt conflict resolution before giving up.
